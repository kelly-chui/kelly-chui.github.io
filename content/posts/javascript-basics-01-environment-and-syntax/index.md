# 환경 설치

## 자바스크립트란?

- HTML과 CSS와 함께 클라이언트 측 웹 프로그래밍을 할 수 있도록 돕는 프로그래밍 언어
- C, C++, 자바와 같은 언어에 비해 문법이 쉽고 브라우저에서 바로 실행됨
- 조금만 배워도 HTML, CSS와 함께 바로 사용 가능

### 컴파일러 언어와 인터프리터 언어

자바스크립트는 인터프리터 언어로 작성과 동시에 바로 실행할 수 있는 언어이다.

- 프로그래밍은 일반적으로 컴파일러와 인터프리터라는 과정을 통하여 프로그래밍 진행
- 바로 작성한 프로그램을 확인할 수 있는 언어를 인터프리터 언어, 컴파일러 언어는 소스 코드를 기계어로 번역하는 과정이 필요함.

### 기반 환경 구축하기

ES6부터 CLI를 활용해야한다. node.js, npm, react, git등을 활용하려면 터미널을 활용해야 한다.

- WebStorm, Visual Studio Code
- node.js, npm, 웹팩, 바벨 등 설치 필요
- 수업은 텍스트 편집기 서브라임텍스트 3와 크롬 개발자 도구 사용

## HTML 기초

`<head>`: 제목 표시줄의 내용

`<body>`: 웹브라우저 환경에서 보여주는 부분

## 기본 문법

### 변수

- 프로그램을 실행하는 도중에 임의 값을 저장해 두고 읽을 수 있는 가상의 공간

### 연산자와 함수

- 숫자를 계산하는 연산자를 산술 연산자
- 함수는 어떠한 입력을 받아 특정한 기능이나 계산을 수행한 다음 그 결과를 돌려주는 역할
- 입력이나 결과 없이도 함수 안에서 특정 기능만 수행할 수 있는 점이 수학에서 쓰는 함수와 다름

### 문자열

- 문자열은 문자에 대한 길이나 두 문자열을 이어 붙이는데 사용함
- 문자열의 길이를 알아내려면 문자열의 길이 속성에 접근할 수 있어야 함

### 배열

- 여러 값이 연속으로 저장된 공간
- 배열을 이용하면 여러 개의 변수를 저장해 두고 인덱스 라고 부르는 주소를 이용해서 접근할 수 있음
